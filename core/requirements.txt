# ── Optional: faster-whisper (MIT) — STT provider AND the Gemini-STT fallback ────
# Two uses, both optional:
#   1) STT_PROVIDER=whisper — run it as the primary (needs CUDA for large-v3 float16).
#   2) STT_FALLBACK=whisper (default) — CPU int8 last-resort when Gemini STT returns
#      nothing. CPU-only, NO CUDA needed; slow on long clips but keeps a transcript.
# Absent → the fallback is simply skipped (pipeline continues transcript-free).
#   pip install faster-whisper>=1.0.0                     # enables the CPU int8 fallback
#   pip install nvidia-cudnn-cu12==8.9.7.29   # only for CUDA float16 (9.x fails — see WHISPERX_GUIDE.md)
